06 — Motion

Motion principles

Subtle, purposeful, and accessible. Respect prefers-reduced-motion.

Page enter

Fade + 8px translateY · 400ms ease-out · once per route.

Scroll reveals

Stagger children 80ms apart on intersection. Disable below 720px.

Hover lift

2–4px translateY + shadow expansion · 200ms. Buttons, cards, nodes.

07 — Accessibility

Non-negotiables

  • ✓ WCAG 2.2 AA across all pages
  • ✓ Color contrast ≥ 4.5:1 body, 3:1 large text — verified on every section
  • ✓ Focus rings: 2px blue, 2px offset — always visible, never outline: none
  • ✓ Skip-to-content link, semantic landmarks, descriptive alt text
  • ✓ Keyboard reachable everywhere; visible focus order matches visual order
  • ✓ Motion respects prefers-reduced-motion
